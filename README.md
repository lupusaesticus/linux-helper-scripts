# linux-helper-scripts
Personal scripts I've created to make my life easier, mostly using Claude or Gemini.

### Scripts
- **yaml_memcheck.sh**: Audits Docker memory limits by scanning a given folder for .yml files.
- **dhealth.sh**: Outputs drive health info to a file to be read and presented by other scripts such as conky
- **organize_files_script.sh**: Turns a flat file structure into folder per file in working dir, useful for organising for Audiobookshelf
